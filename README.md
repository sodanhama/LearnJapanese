# LearnJapanese
## Description
This is a static website I made with HTML, CSS, and JS, to introduce the main Japanese writing systems - Hiragana, Katakana, and Kanji.

This is a barebones website on purpose since I always find a minimal dark mode website easy to read and refer to, and when I was learning Japanese writing systems, I had no clue how to change Duolingo to dark mode.

![Screenshot](https://cdn.hackclub.com/019f3015-0e02-7728-8d68-ea8302bb40cb/screenshot_2026-07-05_at_7.51.42___am.png)

## Features
I used JS and CSS to change the fonts and texts to Japanese on hovering over the texts Hiragana, Kanji, and Katakana.

I utilised grid display to make visually appealing cards for Hiragana, Katakana, and Kanji.

## AI Usage
AI was not used in this project.
